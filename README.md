# Stm32f411-forward-collision-warning  
Hardware Components:  
Central Processing Uni: STM32F411 Microcontroller  
Sensor Module: HC-SR04 Ultrasonic Sensor (For forward obstacle distance measurement)  
Display Module:LED Display (I2C interface) (For real-time distance and vehicle status visualization)  
Actuators & Motor Drive:L298N  + 4 motor  
Visual Alert: Red / Yellow LEDs    
Audio Alert: Active Buzzer  
Power Supply:3x 18650 Li-ion Batteries
