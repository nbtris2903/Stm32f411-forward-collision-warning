# Stm32f411-forward-collision-warning  
Linh kiện sử dụng (Hardware Components):  
Khối xử lý trung tâm:Mạch STM32F411  
Khối cảm biến:Cảm biến siêu âm HC-SR04(Đo khoảng cách vật cản phía trước)  
Khối hiển thị:Màn hình OLED (I2C)(Hiển thị khoảng cách thực tế và trạng thái xe)  
Khối Động cơ:L298N + 4 motor  
Khối cảnh báo: Đèn LED (Đỏ / Vàng) cảnh báo thị giác  
Còi chíp (Buzzer) cảnh báo thính giác  
Nguồn : Pin 18650 (x3)
